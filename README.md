# ChatApp
ChatApp is a real-time chat application built with Node.js and React.js. 
It enables users to communicate with each other in a seamless and interactive way.

## Features

- Real-time chat functionality
- Easy file shating
- Multiple chat rooms
- Emoji support
- Responsive design

## Technologies Used

- Node.js
- Express.js
- React.js
- Socket.io
- HTML/CSS
- Bootstrap
